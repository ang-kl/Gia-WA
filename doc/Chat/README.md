# doc/Chat/

Serial-numbered chat exchange logs. Part of the `CLAUDE-FULL.md` §1 doc
structure.

- **Filename:** `chat-<version>-<date>.md` — version `M_m_p`, date
  `dd_mm_yy-hhmm` (`CLAUDE-FULL.md` §2). Example: `chat-0_1_0-21_05_26-2130.md`.
- **Update cadence:** per chat reply; append-only, no confirmation gate (§9).
- Each reply opens with its serial number `(№ N - DD-MM 'YY HH:MM TZ)` (§3); the
  session counter resets to 1 per chat session (§3.2).
- Every substantive paragraph ends with a `[§X.Y]` paragraph tag (§4).
