# doc/Archive/

Superseded document versions, retained for traceability. Part of the
`CLAUDE-FULL.md` §1 doc structure.

- When a doc is superseded, the prior version moves here **unchanged** — never
  overwritten, never edited (Rule N-3).
- Archived files keep their original `<type>-<version>-<date>.md` filename.
- An archived `CLAUDE.md` is renamed `CLAUDE-<prior-version>-<date>.md` (Rule A-2).
