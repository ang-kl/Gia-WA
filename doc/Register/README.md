# doc/Register/

Project register — open items, known issues, deferred work, decisions. Part of
the `CLAUDE-FULL.md` §1 doc structure.

- **Filename:** `register-<version>-<date>.md` — version `M_m_p`, date
  `dd_mm_yy-hhmm` (`CLAUDE-FULL.md` §2). Example: `register-0_1_0-21_05_26-2130.md`.
- **Update cadence:** per session, if open items change; append-only, no
  confirmation gate (§9).
- Numbered architecture decisions specific to Gia-WA's deviations from
  `BUILD-PLAN.md` live in the sibling `doc/decisions/` folder as ADRs.
- Superseded versions move to `doc/Archive/` unchanged; never overwrite (Rule N-3).
