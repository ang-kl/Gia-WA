# doc/Persona/

Living capability profile of the Human Lead — working style, skill ratings,
preferences. Part of the `CLAUDE-FULL.md` §1 doc structure.

- **Filename:** `persona-<version>-<date>.md` — version `M_m_p`, date
  `dd_mm_yy-hhmm` (`CLAUDE-FULL.md` §2). Example: `persona-0_1_0-21_05_26-2130.md`.
- **Update cadence:** per session, if capability ratings change (§9).
- **Confirmation gate:** G3 (§8).
- Superseded versions move to `doc/Archive/` unchanged; never overwrite (Rule N-3).
