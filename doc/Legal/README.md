# doc/Legal/

Legal notices — disclaimer, licensing, data-source attribution, jurisdiction
notes. Part of the `CLAUDE-FULL.md` §1 doc structure.

- **Filename:** `legal-<version>-<date>.md` — version `M_m_p`, date
  `dd_mm_yy-hhmm` (`CLAUDE-FULL.md` §2). Example: `legal-0_1_0-21_05_26-2130.md`.
- **Update cadence:** per legal change only (§9).
- **Confirmation gate:** G1 + Human Lead legal review (§8, §9).
- The §2 "Data Sources & Attribution" table should cross-reference
  `doc/third-party.yaml` if that manifest exists (`CLAUDE-FULL.md` §16.2).
- Superseded versions move to `doc/Archive/` unchanged; never overwrite (Rule N-3).
